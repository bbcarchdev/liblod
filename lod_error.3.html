lod_error

Determine if an error occurred

Synopsis

#include <liblod.h>

int lod_error(LODCONTEXT *context);

Description

lod_error determines whether an error occurred in a previous call to a liblod function involving the supplied context.

If an error has occurred, a description of the error condition can be obtained with lod_errmsg.

Return value

If no error has occurred, lod_error returns zero.

Otherwise, lod_error returns a non-zero integer.

See also

liblod, lod_errmsg.
